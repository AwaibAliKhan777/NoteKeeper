# Django Notes Management System

## Description
A web application built with **Django** to **create, update, and delete notes efficiently**. It provides an intuitive user interface for quick note management. Supports **100+ notes** with a modern, responsive design.

---

## Features
- Add new notes with title and content  
- Update existing notes  
- Delete notes  
- View all notes in a clean, responsive UI  
- Fully functional **CRUD operations** using Django ORM  

---

## Technologies Used
- **Backend:** Django 6.0  
- **Frontend:** HTML, CSS  
- **Database:** SQLite (default Django database)  
- **Other Tools:** VS Code, Python 3.11
